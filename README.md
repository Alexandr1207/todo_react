# React TODO App (MUI)
Простое и функциональное приложение для управления списком задач, созданное с использованием **React** и библиотеки компонентов **Material UI**.

## Основные возможности

- **Добавление задач**: Быстрое создание новых задач через поле ввода.
- **Управление статусом**: Перенос задач между списками "План" и "Готово" одним кликом по чекбоксу.
- **Редактирование "на месте"**: Возможность изменить текст задачи прямо в списке без открытия дополнительных окон.
- **Удаление**: Полное удаление задач из любого списка.
- **Сохранение данных (Persistence)**: Все ваши задачи сохраняются в `localStorage` браузера — данные не пропадут даже после перезагрузки страницы.
- **Адаптивный дизайн**: Благодаря Material UI приложение отлично выглядит на любых экранах.

## Технологический стек

* **Frontend**: React (Hooks: `useState`, `useEffect`).
* **UI Library**: Material UI (MUI) — использование контейнеров, кнопок, иконок и текстовых полей.
* **Icons**: MUI Icons.
* **Storage**: Браузерный LocalStorage API.

## Структура кода
* **App.js** — основной компонент, содержащий логику управления состоянием (Lifting State Up) и работу с LocalStorage.
* **AddField** — компонент для ввода и валидации текста новой задачи.
* **AddTask** — компонент строки задачи, поддерживающий режимы чтения и редактирования.
